### 자기소개
- **_데이터 분석가_** <br>

학부 때 '데이터베이스' 강의를 통해 우리가 늘 편리하고 당연하게 사용하고 있는 서비스의 뒤편에 가려져 있던 거대한 데이터의 세계에 처음 마주하게 되었고, SQL이라는 언어를 통해 이 방대한 데이터를 원하는 대로 움직여 의미 있는 가치를 만들어내는 과정에서 큰 매력을 느꼈습니다. <br>
이에 저는 주변에 존재하는 무수한 잠재적 데이터를 수집, 저장, 처리, 분석하고 시각화함으로써 끝내 숨겨져 있던 인사이트를 발굴하여 실질적인 가치로 연결하는 일에 기여하고 싶다는 강력한 목표를 가지게 되었습니다. <br>
이후 실제 데이터셋을 직접 수집해보고 python의 numpy, pandas, matplotlib, seaborn 등으로 분석하고 시각화하는 과정을 통해 데이터셋 그 자체만으로는 결코 보이지 않았던 숨겨진 의미를 이끌어내는 경험을 하며 데이터 분석가라는 직무를 더욱 확신하게 되었습니다. <br>
이러한 목표를 설정하는 것에 그치지 않고, 데이터 분석의 기본이 되는 SQL 역량을 다지고자 SQL 개발자(SQLD) 시험에 합격하여 자격증을 취득하였으며, 이를 시작으로 꾸준한 자기개발을 이어나가고 있습니다.
